Конвертер оборудования → it-assets

Загрузите .xlsm/.xlsx с листами «ОС», «Мелочи», «История» — получите готовые CSV для импорта. Всё работает прямо в браузере, файл никуда не отправляется.

Перетащите файл сюда или нажмите, чтобы выбрать
.xlsm / .xlsx
Как импортировать в it-assets:
1. Настройки → Импорт → загрузите itassets_import_assets.csv (оборудование)
2. Затем загрузите itassets_import_history.csv (история перемещений)
Тип файла приложение определяет само по заголовкам.
